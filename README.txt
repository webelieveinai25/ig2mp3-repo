Instagram to MP3 Converter
==========================

Convertitore di video Instagram in MP3 con interfaccia grafica e riga di comando.

USO RAPIDO:
1) Doppio click su ConvertiIG2MP3.exe
2) Incolla i link Instagram o carica links.txt
3) I file MP3 saranno salvati in output_mp3/

OPZIONI AVANZATE:
- CLI: python converti_instagram_mp3.py --url https://www.instagram.com/reel/XXXX/
- Batch: python converti_instagram_mp3.py --links links.txt --out cartella_output
- Qualit√†: --quality 320 (128/192/256/320 kbps)

REQUISITI:
- ffmpeg.exe (incluso nel pacchetto)
- Connessione internet

SUPPORTO:
- Formati: Reel, Post, Stories Instagram
- Output: MP3 con metadati
- Log: errors.log e report.csv per debug
